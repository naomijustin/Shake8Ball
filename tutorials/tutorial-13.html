13: Toast and modal for 'How To' & 'Disclaimer'

Now was to add the disclaimer and the how to toasts and modals.

I didn't want to take away from the aesthetic of the page but wanted to add information about the site's disclaimer and a 'How To' guide if the user needed assistance.

I decided to add these via toasts that would lead to more details, the toast then disappearing after a period of time. I also added icons that would always be static on the page to make sure that the user would be able to access this information at a later point if they wanted to.

Adding the toasts and modals

Specifically for the 'How To', I wanted the user to be able to figure it out themselves as that can be part of the fun, so I delayed the 'How To' toast to only appear after 8 seconds if the user had not yet clicked on the ball.

                                
    setTimeout(function(){
        if (!firstShakeCompleted) {
            showHowToToast();
        }       
    }, 8000);

This resulted in the below toasts appearing at different times to not overwhelm the user having the toasts show up all at once.

To create and add the icons to trigger the Disclaimer and How To modal once they had closed, I decided the best would be a low opacity yet easily identifiable set of icons.

I compared the Font Awesome icon set however ultimately went with the Google Font icons as they felt cleaner, less bulky and less invasive - as to not take away from the overall feel of the site.

Resolving mobile issues

One other issue that was found, was that for mobile users, if they tried to close the toast but did not hit the 'X' on the first tap, the white ball would 'follow' the tap, ie. be facing downwards and would stay like this until the ball shaked.

So to avoid this, I reset the white 8 ball position if the toast was tapped on but not closed.

                                
    $('#toastTipPrompt, #toastHowToPrompt').click(function(){
        $('#image8BallWhiteCircle').css('top', '50%');
        $('#image8BallWhiteCircle').css('left', '50%');
        $('#image8BallWhiteCircle').css('transform', 'translate(-50%, -50%)');
    });
