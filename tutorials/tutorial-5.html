5: Add Cloud and Animate

Now to create the cloud. The goal for this was to create a magical cloud that would animate below and behind the 8 ball that would add to the mystical element of the site.

Creating the cloud image

I created the cloud image in Figma using the pen tool to create an organic shape, and then used a vertical gradient to give a magical element to the cloud.

I created it to have 100% opacity at the top, 0% opacity at the bottom to create the gradient, and then for the entire image, lowered it to 49% for the overall look.

Once that was created, I exported it out from Figma as an SVG, again adding some blank area at the top to compensate for the centering so it would be positioned below the 8 ball.

Adding in the cloud image

After adding in the cloud image, I again centered it the same as the other image that have been added so far (see Tutorial 2).

Then, specifically to the div wrapping this cloud, I added in the below to scale the image, but to also position it on the right layer relative to the 8 ball.

                                
    z-index: 2;   
    height: 965px;
    max-height: 95vh;

Then, on the image itself, I added the following CSS to also position it correctly inside the wrapping div.

                                
    max-height: 95vh;
    position: relative;
    overflow: hidden;

Animating the Cloud

Now to animate the cloud.

I decided for animating the cloud, the easiest way to create the effect I wanted was to use a combination of skewing, scaling and rotating the image.

To create an infinite loop of the effect, I set both the 0% and 100% to be the same so that there was no jump cut/jitter to the animation when restarting. I then played around with setting slight changes to each element of the transform until I was satisfied. The below CSS keyframes is what I ended up deciding to move forward with.

                                
    @keyframes cloudFloat1 {
        0% {       
            transform: skew(3deg, 3deg) scale(1.04) rotateY(3deg);       
        }
        25% {       
            transform: skew(-3deg, -3deg) scale(0.97) rotateZ(-3deg);       
        }
        50% {       
            transform: skew(5deg, 5deg) scale(1.04) rotateX(10deg);       
        }
        75% {       
            transform: skew(-3deg, -3deg) scale(0.97) rotateZ(0deg);
        }   
        100% {       
            transform: skew(3deg, 3deg) scale(1.04) rotateY(3deg);       
        }
    }

Now that the animation has been created, next is to set it to the image itself, along with the duration, the count and in the specific case, the timing function as it helps add to the transitions between each stage.

                                
    animation-name: cloudFloat1;
    animation-duration: 5s;
    animation-iteration-count: infinite;
    animation-timing-function: ease-in-out;
