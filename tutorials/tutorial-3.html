3: Adding more images

Now that we have the background scaled correctly and responsive, we can now start adding more images such as the 8 ball layers.

My goal was to be able to have the white circle of the 8 ball rotate and follow the user's mouse when moving around but to not go too far out of the dark circle to not lose the immersion of this effect.

Creating the layers

Before we can consider the functionality of how this can be done, we first need to set up the layers correctly to allow this to be done later.

To do this, I created 3 separate layers for the ball:

  1. The 8 ball base
  2. The white circle with the 8 inside
  3. The shine of the 8 ball

The “8 ball base” purpose is to be the base of the ball itself and be in a fixed position.

The “white circle with the 8 inside” purpose is to follow the mouse cursor around to create the illusion that the ball is rotating and following the mouse.

Finally, the “shine of the 8 ball” purpose is to be a fixed semi transparent layer on top of both of these to contribute to the illusion of a 3D object. On the movement of the white circle, the white circle and 8 will travel underneath this layer to create a shine over the top.
