4: Text and Dropshadow

Next is to add the circular text around the 8 ball. There are a few different methods that could be used to achieve this, one being to create a circular path and for the text to follow this path, however due to the additional drop shadow colour to consider, I decided to create a circular image in Figma instead and then resize the image to the viewport accordingly.

Creating the text

I added the text and font that I wanted into the Figma project, and then used the Figma 'To Path' plugin to create the curved text that would wrap around the 8 ball.

I then duplicated the curved text, changed the colour to the drop shadow colour and put it behind the first to create the drop shadow curved text.

I then grouped these with some transparent space to compensate for the centering (as it would be sitting above the 8 ball, rather than centered with the 8 ball). I exported these as a single SVG image so that they could be infinitely scaled without resolution loss and added it to the site.

Adding the text

I then centered it, again using the same CSS as previously used with the other images (see Tutorial 2).

I then added an addition just for the text to scale larger than the 8 Ball so that it would be positioned vertically above on the page, and to also make sure that the text would also be a layer above the 8 ball so the text would not be covered if there was an issue on a smaller device.

                                
    z-index: 7;   
    max-height: 87vh;

This resulted in the text positioned above the 8 Ball and wrapping around in an arc.
